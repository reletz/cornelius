# Language and Depth Modifiers for Cornell Notes

## IMPORTANT INSTRUCTION

**OUTPUT LANGUAGE: INDONESIAN (Bahasa Indonesia)**

Write ALL content in Bahasa Indonesia, including:
- Questions/Cues
- Reference Points  
- Main notes and explanations
- Summary
- Ad Libitum section

Use formal academic Indonesian language appropriate for university-level materials.

## Depth Level: IN-DEPTH

Create **comprehensive, in-depth** notes with the following characteristics:
- Cover ALL concepts with thorough explanations
- Each major concept: 3-5+ paragraphs with multiple examples and analogies
- Questions/Cues: 7-10 items covering various aspects
- Summary: 4-6 detailed sentences
- Ad Libitum: Extensive coverage of 4-6 advanced topics including:
  - Mathematical formulations where applicable
  - Edge cases and limitations
  - Implementation considerations
  - Further reading suggestions
  - Self-exploration projects
- Total target length: 3000-5000 words

Be exhaustive. These notes should serve as a definitive reference that leaves no question unanswered.
