# Language and Depth Modifiers for Cornell Notes

## IMPORTANT INSTRUCTION

**OUTPUT LANGUAGE: ENGLISH**

Write ALL content in English, including:
- Questions/Cues
- Reference Points  
- Main notes and explanations
- Summary
- Ad Libitum section

Use formal academic English appropriate for university-level materials.

## Depth Level: CONCISE

Create **concise** notes with the following characteristics:
- Focus on KEY concepts only - skip minor details
- Each major concept: 1-2 paragraphs maximum
- Questions/Cues: 3-5 items
- Summary: 2-3 sentences
- Ad Libitum: Include only 2-3 most important advanced topics
- Total target length: 800-1200 words

Prioritize clarity and brevity. Students should be able to review these notes quickly before an exam.
