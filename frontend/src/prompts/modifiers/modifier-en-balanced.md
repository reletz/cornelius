# Language and Depth Modifiers for Cornell Notes

## IMPORTANT INSTRUCTION

**OUTPUT LANGUAGE: ENGLISH**

Write ALL content in English, including:
- Questions/Cues
- Reference Points  
- Main notes and explanations
- Summary
- Ad Libitum section

Use formal academic English appropriate for university-level materials.

## Depth Level: BALANCED

Create **balanced** notes with the following characteristics:
- Cover all important concepts with adequate detail
- Each major concept: 2-4 paragraphs with examples
- Questions/Cues: 5-7 items
- Summary: 3-5 sentences
- Ad Libitum: Include 3-4 advanced topics with moderate detail
- Total target length: 1500-2500 words

Balance thoroughness with readability. These notes should serve as a complete study resource.
